## 객체 지향 프로그래밍 (OOP, Object-Oriented Programming)

- 실제세계를 객체라는 단위로 나누고 **객체들간의 상호작용**을 의미한다.
- 즉, 프로그램을 여러개의 독립된 단위인 객체들의 모임으로 파악하고자 하는 것이다.
- **자바는 대표적인 객체지향 프로그래밍 언어이다.**
